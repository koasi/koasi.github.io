# **App Name**: Pomodoro Flow

## Core Features:

- Countdown Timer: Start, pause, and reset countdown timer functionality.
- Mode Switching: Switch between Pomodoro (25 min), Short Break (5 min), and Long Break (15 min) modes.
- Custom Settings: Allow users to adjust the duration of each mode.
- Sound Notifications: Play a sound when the timer ends (with on/off switch).
- Dynamic Title Display: Show the remaining time in the browser tab title.
- To-Do List: Simple task addition, completion, and deletion functionality using local storage.
- Progress Tracking: Basic statistics on completed Pomodoro sessions using local storage.

## Style Guidelines:

- Primary color: Soft pastel orange (#FFB347), evoking focus and warmth, without being as bold as pure orange. This matches the tomato theming but with softer energy.
- Background color: Very light pastel orange (#FFF3E0). Subtle and calm.
- Accent color: Light coral (#FF8A65), creating contrast for CTAs and highlights.
- Body and headline font: 'PT Sans' for a balance of modern style and readability.
- Use clean and minimalist icons for timer controls and settings.
- Mobile-first, single-column layout for phones; adaptable dual-column layout for tablets/desktops (timer on left, task list on right).
- Subtle animations for timer transitions and task completions.
